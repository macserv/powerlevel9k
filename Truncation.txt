P9K_DIR_SHORTEN_STRATEGY="en_vowel"


Strategy
--------
Attempt to position delimiter at the last non-consonant character.
If not found, truncate before the last character.

If P9K_DIR_SHORTEN_KEEP_EXT is true, truncate name and extension separately, and include the "." separator.

If P9K_DIR_SHORTEN_KEEP_EXT is true:
    P9K_DIR_SHORTEN_LENGTH     is the desired maximum presented characters of the "name" part of the path component (not counting the delimiter).
    P9K_DIR_SHORTEN_EXT_LENGTH is the desired maximum presented characters of the "extension" part (not counting the delimiter).
Else:
    P9K_DIR_SHORTEN_LENGTH is the desired maximum presented characters of the entire path component (not counting the delimiter).

A minimum of two characters must precede the truncation delimiter.
A minimum of one character  must follow  the truncation delimiter.
If the extension component is truncated separately, only one character must precede its delimiter.

According to the above rules, the lowest honored value for P9K_DIR_SHORTEN_LENGTH is 3, and the lowest honored value for P9K_DIR_SHORTEN_EXT_LENGTH is 2.


Notes
-----
Truncation threshold = shortening length + delimiter length
Truncation marker set: [AaEeIiOoUu\W\_] All vowels, all non-word characters, underscore (considered a "word character")
